### set
```text
在Python中，set 类型的内部实现通常基于哈希表（也称为散列表），但严格来说，我们不能直接称 set 为哈希表。
set 是一种无序的、不包含重复元素的数据结构，它利用哈希表来高效地实现元素的存储、查找、插入和删除操作。
哈希表是一种通过哈希函数将键（key）映射到表中的一个位置来访问记录的数据结构。
在Python的 set 中，每个元素都可以看作是哈希表的键，但不需要与之关联的值（因为 set 只存储元素本身）。
当你向 set 中添加、删除或查找元素时，Python 内部的哈希表机制会利用哈希函数来快速定位这些操作的位置。
虽然 set 内部使用哈希表来实现其功能，但Python的官方文档或标准库文档中并没有明确将 set 定义为哈希表。
这是因为 set 是一种高级抽象，它隐藏了底层实现的细节，使得用户能够专注于 set 提供的功能，而无需关心它是如何实现的。
总结来说，Python的 set 确实使用了哈希表作为其底层实现的一部分，以提供高效的元素管理功能，
但通常我们不会直接称 set 为哈希表。
```
